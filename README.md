# Mutual Fund Analytics & Risk Intelligence Dashboard

## Project Overview

The Mutual Fund Analytics & Risk Intelligence Dashboard is an end-to-end data analytics project developed to analyze mutual fund performance, investor behavior, industry trends, and risk-adjusted returns. The project combines data engineering, financial analytics, and business intelligence to transform raw mutual fund data into actionable insights.

Using Python, SQLite, SQL, and Power BI, the platform enables comprehensive analysis of fund performance, risk metrics, investor participation, and industry growth trends.

---

## Project Objectives

* Build a robust ETL pipeline for mutual fund datasets.
* Clean and validate financial data from multiple sources.
* Design and implement a SQLite database for analysis.
* Perform Exploratory Data Analysis (EDA) to identify trends and patterns.
* Calculate performance and risk metrics such as Sharpe Ratio, Sortino Ratio, Alpha, Beta, VaR, and CVaR.
* Develop an interactive Power BI dashboard for data visualization and decision support.
* Generate meaningful insights and recommendations for investors and analysts.

## Power BI Dashboard

The dashboard consists of four interactive pages:

### Industry Overview

Provides a high-level view of industry performance including AUM, SIP inflows, folios, and scheme statistics.

### Risk Analytics

Analyzes downside risk using VaR, CVaR, Maximum Drawdown, and Rolling Sharpe Ratio.

### Fund Analysis

Compares fund performance across categories using risk-adjusted metrics and performance indicators.

### Investor Analytics

Examines investor behavior, transaction trends, and geographical participation patterns.

---

## Key Findings

* SIP inflows showed consistent growth throughout the analysis period.
* Equity-oriented funds dominated industry participation and AUM.
* Risk-adjusted performance varied significantly across fund categories.
* Small-cap funds exhibited higher volatility and downside risk.
* Investor participation was concentrated in major states and urban regions.
